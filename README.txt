Un plugin crée pendant un stage pour aller avec GLPI

Il permet de parcourir la base de données tout les jours et de passer les tâches d'un ticket qui sont à l'état 0 (information), à l'état 1 (à faire)

N'oubliez pas de faire un "composer install" pour installer les librairies requises

Modifiez-le à votre guise, il y a très peu de chances que ce plug-in soit mis à jour !

N'utilisez le fichier web.php qu'en cas de tests !
